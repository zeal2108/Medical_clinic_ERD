# Medical_clinic_ERD
A Practice Entity Relation Diagram (ERD) with cardinality (Crow's Feet Notation) for a small medical clinic's Database

- The repository contains an exported image from eraser.io which was used to design the ERD.
- The ERD follows PostgreSQL syntax wherever possible.
